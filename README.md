# Motion-Picture-Finder

## Description 

Show Finder will display the TV show title, a picture of the show, and the streaming servicesâ€™ logo where the show is available. All that you need to do is type in your desired show title and hit search. When you click on a logo, the site will direct you to your show on the chosen site. Logo icons linked to websites with more information on the show are also available. Show Finder will also create a search history, so you can refer to the streaming location of titles you have searched before.


## Technologies Used

- HTML/CSS
- JavaScript
- Local storage
- Bulma CSS Framework
- Utelly API: gets information from services like Netflix, Amazon Prime Video, iTunes, Apple TV+, etc.
- MDB API: 
- Slack & Zoom for communications
